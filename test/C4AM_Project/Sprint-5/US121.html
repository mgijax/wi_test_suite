SNP out of sync mode

  • User Story #121 - Implement out of sync SNP mode for Prod WI

Test Requirements

  1. SNP associations to markers are limited to the dbSNP associations
  2. there should be no "within coordinates" associations
  3. there should be no ""+/- n kb associations"
  4. verify certain fields on the SNP QF are inactive
  5. verify no links to GBrowse/Marker Summary at the top of the SNP results page when there's a coord search

Manual tests

  1. SNP Query form renders in out of sync mode

    new test

    1. Genome Coordinate Discrepancy text appears at top of query form
    2. SNPs Located Near Genes section of query form is disabled and displays a discrepancy message
  2. SNP Summary renders in out of sync mode

    new test

    1. Query for Gene Symbol/Name = Qk
      • Genome Coordinate Discrepancy text appears at top of query summary
      • No SNPs are returned with a function class of N bp upstream
      • No SNPs are returned with a function class of N bp downstream
      • No SNPs are returned with a function class of within coordinates
    2. Query for Coordinate range = Chr5:75000000-76052747 and select strain: C57BL/6J
      • Genome Coordinate Discrepancy text appears at top of query summary
      • No link to the Marker Summary or GBrowse should be present at the top of the SNP Summary
      • No SNPs are returned with a function class of within coordinates
  3. Display of SNP associations render correctly on the Marker Detail in out of sync mode
