Update marker detail page to give an error message if non-numeric characters are entered in the Flank field.

  • User Story 153, Update marker detail page to give an error message if non-numeric characters are entered in the Flank field.

Test Requirements

Production marker detail page:
  • Verify that a non-numeric value for Flank results in a helpful error message.
  • Verify that a float (non-integer) value for Flank results in a helpful error message.
  • Verify that a valid integer value for Flank results in a successful return of sequence data.
Public marker detail page:
  • Verify that a non-numeric value for Flank in the Sequence Map section results in a helpful error message.
  • Verify that a float (non-integer) value for Flank in the Sequence Map section results in a helpful error message.
  • Verify that a non-numeric value for Flank in the Sequences section results in a helpful error message.
  • Verify that a float (non-integer) value for Flank in the Sequences section results in a helpful error message.
  • Verify that a valid integer value for Flank results in a successful return of sequence data.

Tests

Manual tests

Production marker detail page:
  1. Verify that a non-numeric value for Flank results in a helpful error message.

    jlewis - 10/10/12

    1. Display the marker detail page for Kit (MGI:96677 ).
    2. In the Sequences section, check the checkbox for the genomic sequence, enter "Hi" in the Flank box, and click the "Go" button.
        Expected results: a box will pop up with a descriptive error message.

    jlewis - 10/10/12

    1. Display the marker detail page for Kit (MGI:96677 ).
    2. In the Sequences section, check the checkbox for the genomic sequence, enter "2 " (a 2 followed by a space) in the Flank box, and click the "Go" button.
        Expected results: a box will pop up with a descriptive error message.
  2. Verify that a float (non-integer) value for Flank results in a helpful error message.

    jlewis - 10/10/12

    1. Display the marker detail page for Kit (MGI:96677 ).
    2. In the Sequences section, check the checkbox for the genomic sequence, enter "2.0" in the Flank box, and click the "Go" button.
        Expected results: a box will pop up with a descriptive error message.
  3. Verify that a valid integer value for Flank results in a successful return of sequence data.

    jlewis - 10/10/12

    1. Display the marker detail page for Kit (MGI:96677 ).
    2. In the Sequences section, check the checkbox for the genomic sequence, enter "2" in the Flank box, and click the "Go" button.
        Expected results: sequence data for Kit with 2 kb extra at the beginning and end of the sequence.
