Curators can run the Coordinate Loader/h1>
  • User Story #19 - Curators can run the coordinate loader without SE help

Test Requirements

  1. A how-to guide is available to the curators for this load process.
  2. Curators can schedule the Coordinate Loader to run once per week with a single input file.
  3. QC Checks are run as part of the load and the curator can review the QC reports after the load runs.
  4. This process can be done independently without any help from the software group.
