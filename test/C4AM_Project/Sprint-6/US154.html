Load Feature Types for new markers

  • User Story 154, Load Feature Types (mcv) for new markers

Test Requirements

  • Load feature types for all new B38 markers
  • Display in the test-Prod EI
  • Display in the test-Prod WI - Marker QF Feature Type table (?) and Marker Detail

Tests

Manual tests

  1. Verify feature types (MCV annotations) are present for all new markers (using the EI)

    new test

    1. Query for feature type lincRNA, using the appropriate creation date
      • Expected results:8 returned
    2. Query for feature type protein coding, using the appropriate creation date
      • Expected results:547 returned
    3. Query for feature type pseudogene, using the appropriate creation date
      • Expected results:246 returned
    4. Query for feature type pseudogenic gene segment, using the appropriate creation date
      • Expected results:2 returned
    5. Query for feature type unclassified gene, using the appropriate creation date
      • Expected results:385 returned
    6. Query for feature type unclassified non-coding RNA gene, using the appropriate creation date
      • Expected results:3 returned
  2. Verify feature types (MCV annotations) are displayed on the Prod WI - Marker Detail page for the new genes

    new test

    1. Verify Gm21770
      • Expected results:Feature Type = lincRNA
    2. Verify Gm21793
      • Expected results:Feature Type = protein coding
    3. Verify Gm21917
      • Expected results:Feature Type = unclassified non-coding RNA gene
  3. Verify feature types (MCV annotations) are displayed on the Prod WI - Marker Query Form (?)

    new test

    1. Select the Marker QF and show all the Feature Types
      • Expected results:See counts in test 1 and verify counts have increased accordingly
