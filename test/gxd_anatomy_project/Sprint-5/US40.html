Add Annotation Count to Anatomy Tree

  • User Story 40 - Add Annotation Count to Anatomy Tree

Contents: Title | Test Requirements | Manual Tests | SQL Tests

Test Requirements

  1. Display the number of GXD results for an EMAPA or EMAPS term in the tree view of the new Anatomy Browser
    • Annotations to the EMAPS term are driven by the AD/EMAPS mapping table.
    • Annotation counts for an EMAPA term is inferred from the associated EMAPS terms
  2. Link the number to the GXD Summary page; Results tab.
  3. The count of GXD results includes the annotations to each term plus all of its children.
  4. When zero results are associated to a term - display a 0 count -- but don't make it a link.
  5. When 1 result is associated to a term - remove pluralization in the text associated to the link.
