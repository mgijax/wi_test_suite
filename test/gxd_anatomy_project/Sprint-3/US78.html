Add Synonyms to Anatomy Browser Search Pane

  • User Story 78 - Add display for matching synonyms to anatomy browser

Contents: Title | Test Requirements | Manual Tests | SQL Tests

Test Requirements

  1. Display matching synonym of term when it is the ONLY reason the term is returned.
    1. If more than 1 synonym are matched in this case, select one at random for display.
  2. Display the term followed by the synonym in the results.
    1. Enclose the synonym in parentheses
    2. Terms are linked in the results - synonyms are not part of the link
    3. Bold the matching text in the synonym
  3. Display consistently in all supported browsers (IE, Firefox, Safari)

Tests

Manual tests

  1. Display synonyms when appropriate in matching text

    PASS - jlewis 11/12/2013

    1. Search for text that matches both term and a single synonym; prim . Expect: no synonym display
    2. Search for text that matches both term and multiple synonyms; endoderm . Expect no synonym display
    3. Search for text that matches a single synonym for a term; another. Expect: synonym to display
    4. Search for text that matches multiple synonyms for a term; fake. Expect: 1 of the synonyms to display
    5. Search with text that does not match any synonyms, but returns results; renal. Expect: no synonyms displayed in results
    6. Search with text that does not match any synonyms and doesn't return any results; football. Expect: no results
    7. Search for a synonym with special characters; xxxxx. No example for this test case in test data.
  2. Return synonyms with matching text in bold, in parens, and not part of link for all supported browsers
