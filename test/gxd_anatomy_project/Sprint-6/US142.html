Add filters to the GXD Result Summary

  • User Story 142 - Add filters to the GXD Result Summary

Contents: Title | Test Requirements | Manual Tests | SQL Tests

Test Requirements

  1. Add filter option for selected fields on the GXD Results Summary to provide the user an additional way to narrow down results.
    1. Anatomical Structure
    2. Assay Type
  2. The filter options are present at the top of the GXD Results Summary, but should filter all 4 GXD Summaries: Gene, Assay, Result, Image
  3. Filters work on the "bucket" of data returned from the Query Form search.
    • Rerunning the Query Form search with new or the same parameters removes all filters -- resets the "bucket" of data.
  4. Filters list only items that are present in the current "bucket" of data.
    • The original list of filter items persists through all filter events
  5. Users may select 1 or more items for each filter available.
  6. User may un-select 1 or more items for each filter available.
    • Provide two options: button at top of page or un-check of box within Filter
  7. Provide users the option to remove all filters in 1 step.
  8. Provide users with updated counts each time a filter is added/removed
    • Applicable for all 4 GXD Summaries: Gene, Assay, Results, Image
  9. Pagination counts and controls are updated after each filter event for all GXD Summaries
  10. You searched for values and original query counts continue to be displayed during all filter events.
  11. Use the Reference QF as a guide to the UI display as applicable.
  12. Filters should work properly in all supported browsers (IE, Firefox, Safari)
