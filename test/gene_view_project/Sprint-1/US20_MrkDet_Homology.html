Marker Detail - Homology ribbon

  • US 20 - Marker detail: new homology ribbon (combine vertebrate homology and human homolog ribbons)

Test Requirements

  1. Combine the 2 ribbons: Vertebrate homology and Human homologs into 1 ribbon (with 2 sections)
  2. Display the fields for the human homologs at the top of the ribbon. Include the following fields:
    1. Human Orthologs: human symbol, human name, orthology source
    2. Links: HGNC, NCBI, neXtProt AC, HCOP (?)
    3. Human Synonyms
    4. Human Chr (Location)
  3. When no human orthologs are associated to the mouse gene none of these labels are displayed.
  4. When multiple human orthologs are associated to the mouse gene repeat these fields for each one.
  5. Display the fields for the Vertebrate homology:
    1. HomoloGene: link to HomoloGene Homology Class page, list of organisms/counts in HomoloGene class
    2. HCOP: link to HCOP for all human genes associated to mouse marker (?)
    3. Protein SuperFamily: link
    4. Gene Tree: link
  6. When data fields don't exist for a marker -- do not display the label
