Test Requirements

Richard has written a detailed requirements document for the conversion to NCBI Blast. This list of requirements is limited to the Marker Detail work in US 2. It is a summary level list used to guide creation of test cases. See the document for implementation details. (add link here)
  1. Convert the "forward to MouseBlast" option in the Sequence ribbon to "forward to NCBI BLAST"
  2. Convert the check boxes in the Sequence ribbon to radio buttons (limit to 1 sequence at a time)
    • Note: this also will limit the FASTA download to 1 sequence at a time
  3. Link directly to NCBI Blast -- no longer use the "Forward to MouseBLAST" page (retire)
    • Create new tab or window? (Richard will ask sequence group)
  4. Forward sequences by accession id
    • Sequence accession ids valid in NCBI BLAST URL
      1. GenBank (LogicalDB=9)
      2. SWISS PROT (13)
      3. RefSeq (27)
    • URL structure/parameters per requirements document (link above)
    • Any error (e.g. not a valid id) is handled by NCBI Blast interface.
  5. Forward gene model sequences by FASTA format using coordinates and genome assembly files (nib files)
    • Types of Gene Models in MGI
      1. NCBI Gene Model (LogicalDB=59)
      2. Ensembl Gene Model (60)
      3. VEGA Gene Model (85)
    • Flanking sequence
    • FASTA format
    • Errors
  6. Forward sequences by FASTA format from EMBOSS
    • Types of sequence accession ids to find in EMBOSS
      1. TrEMBL (LogicalDB=41)
      2. Ensembl Transcript (133)
      3. Ensembl Protein (134)
      4. VEGA Transcript (131)
      5. VEGA Protein (132)
      6. DFCI (35)
      7. DoTS (36)
      8. NIA Mouse Gene Index (53)
    • FASTA format
    • Errors
