Marker Detail - new top ribbon

  • US 17 - Marker detail: combine nomen, synonyms, feature type into one ribbon

Test Requirements

  1. Combine the current Nomenclature, Synonyms, and Feature Type ribbons into 1 new ribbon.
  2. List fields in 2 columns with styling like the Allele Detail page.
  3. Only include labels if values exist.
  4. Display in "open" state -- options to display ribbon in open/close mode NOT part of this user story.
  5. List the following fields in column 1 (when they exist)
    1. Symbol (larger font; bold text) - display "Interim" (e.g. Selk)
    2. Name - wrap for long name
    3. Synonyms - comma delimited list (wrap for long list); optional
  6. List the following items in column 2 (when they exist)
    1. Feature type - assume only 1 type per marker for this user story
    2. MGI ID
    3. Cluster members - comma delimited teaser list of markers plus link with pop-up; optional
    4. Member of - link to cluster marker; optional
  7. Below both columns display BioType Conflict and/or Strain-Specific Marker icons with pop-ups; optional
