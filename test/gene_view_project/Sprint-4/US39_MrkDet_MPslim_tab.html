Implement slimgrid tab summary for MP

  • US 39 - Implement slim grid tab summary for MP

Test Requirements

  1. Clicking on a colored cell results in a new tab with the HMDC-like grid for a gene & slim term combo.
    • This behavior is an exception to the typical links to MGI that remain in the same browser window/tab.
  2. Each link will bring up a new tab (so they may be compared by users).
  3. The heading includes the Gene symbol + MP slim term.
  4. The legend text includes strain background ! (exclamation pt) and normal data (asterisk)
  5. New - add explanation of colored cells.
  6. The summary page is a grid that behaves like the HMDC grid drill down pages
    1. genoclusters are listed (row)
    2. MP terms annotated to the genoclusters are columns
    3. normal data is displayed -- even though they are not in the slim grid itself
    4. each row links to a genocluster page
  7. link to IMSR at top - same as on HMDC
  8. The summary page should act and display correctly in all supported browsers.

Tests

Manual tests

  1. Verify all summaries for a gene with normals, background strains, complex genos, etc.

    new test

    1. Display summaries for each MP slim term and compare to HMDC results; Fmr1. Expect: summaries to be the same as HMDC.
  2. Verify multiple summaries can be viewed at once

    new test

    1. Display multiple summaries; Lep. Expect: each summary to open in a new window -- tab header title?
  3. Verify summary with Normal annotations (asterisks)

    new test

    1. Display marker/term summary w/ a single normal annotation; Bbs1/mortality/aging. Expect: 1 = normal; 1 = rest
    2. Display marker/term summary w/ a single normal annotation; Mus81/cardiovascular system. Expect: 1 = normal; 23 = rest
    3. Display marker/term summary w/ many normal annotation; Ptprcap/immune system. Expect: 2 = normals; 5 = rest
  4. Verify summary with differences in background strains (exclamation marks)

    new test

    1. Display marker/term summary w/ a single strain variation annotation; a/mortality and aging. Expect: 1 = strain variation; 11 = annotations; 1 normal
    2. Display marker/term summary w/ many strain variation annotations; Fmr1/nervous system. Expect: 2 = strain variations (same term; annotated twice); n = annotations
    3. Display marker/term summary w/ both normals and strain variations; Sry; reproductive system. Expect: 27=reg annotations; 2 norms; 1 w/bground sens.
    4. Display marker/term summary with a single annotation that has background sensitivty; Adrb1/mortality and aging. Expect: 1 annotation with an exclamation point.
  5. Verify summary with and without IMSR link - same IMSR link on Summary pages as on the Marker Detail

    new test

    1. Display marker with 1 IMSR strain; Pax9/immune system. Expect: count=1.
    2. Display marker with many IMSR strains; Kcp/mortality and aging. Expect: count=4.
    3. Display marker with 0 IMSR strains; Bmp15/reproductive systems.
    4. Display marker/slim term combo with multiple links to IMSR; Ren1; behavior/neurological. Expect: Ren1=5 imsr strains and Tg(Alb1-Ren)2Unc/0=1
  6. Verify summary factors out complex genotypes

    new test

    1. Display marker/term summary w/ a simple and complex annotation to the same term; Ppt1/nervous system. Expect: complex genotype not listed in slim grid tab summary (term=brain atrophy)
    2. Display marker/term summary w/ an annotation to a term via a complex genotype; Elane/immune system. Expect: Complex genotype and its annotation not displayed; term=decreased susceptibility to endotoxin shock
  7. Verify grid works on all supported browsers: Firefox, Chrome (PC, Mac); Safaric (Mac)
