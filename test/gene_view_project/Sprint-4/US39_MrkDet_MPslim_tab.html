  • US 39 - Implement slim grid tab summary for MP

Test Requirements

  1. Clicking on a colored cell results in a new tab with the HMDC-like grid for a gene & slim term combo.
    • This behavior is an exception to the typical links to MGI that remain in the same browser window/tab.
  2. Each link will bring up a new tab (so they may be compared by users).
  3. The heading includes the Gene symbol + MP slim term.
  4. The legend text includes strain background ! (exclamation pt) and explanation of colored cells.
  5. The summary page is a grid that behaves like the HMDC grid drilldown pages
    1. genoclusters are listed (row)
    2. MP terms annotated to the genoclusters are columns
    3. normal data is excluded (this is different than HMDC)
    4. each row links to a genocluster page
  6. link to IMSR at top - same as on HMDC
  7. The summary page should act and display correctly in all supported browsers.
