Implement slim grid popup for Anatomy

  • US 40 - Implement slim grid pop-up for Anatomy

Test Requirements

  1. Clicking on a colored cell (blue or dog-earred) generates a pop-up window (white cells are not clickable).
  2. The pop-up has an initial set size.
  3. Multiple pop-ups may be displayed by the user.
  4. The pop-up has a blue header bar instead of the standard MGI toolbar.
  5. Each pop-up contains info at the top that identifies the cell clicked -- Gene symbol and EMAPA term (format TBD)
  6. The body of the pop-up is the GXD Summary page with the Matrix x Stage tab open.
  7. The legend is displayed by default.
  8. The summary is the same as would be displayed via the QF for the marker and EMAPA term
    1. The topmost term will be the one clicked on in the slim grid.
    2. One level below will be displayed (i.e. the term is "open")
    3. All results for that EMAPA term will be shown in the matrix: absents and mutants as well as positive and wild type.
  9. All the options available for the GXD Summary are present in the pop-up -- other tabs, filters, downloads, sorts, etc.
  10. The pop-ups should display correctly in all supported browsers.
