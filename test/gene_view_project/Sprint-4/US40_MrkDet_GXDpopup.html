Implement slim grid link outs for Anatomy

  • US 40 - Implement slim grid new tabs for Anatomy

Test Requirements

  1. Clicking on a colored cell (blue or dog-earred) results in a new tab with the Tissue x Stage matrix for a gene & slim term combo.
    • This behavior is an exception to the typical links to MGI that remain in the same browser window/tab.
  2. Each link will bring up a new tab (so they may be compared by users). Tab title = gene symbol + slim term + "gene expression"
  3. The heading is similiar to the Marker/Assay summaries: main heading - gene stuff; sub heading - slim term (but no parens). GXD gold rules the day.
  4. The legend is displayed by default.
  5. The summary is the same as would be displayed via the QF for the marker and EMAPA term
    1. The topmost term will be the one clicked on in the slim grid.
    2. One level below will be displayed (i.e. the slim term is "open")
    3. All results for that EMAPA term will be shown in the matrix: absents and mutants as well as positive and wild type.
  6. All the options available for the GXD Summary are available -- other tabs, filters, downloads, sorts, etc.
  7. The summary page should act and display correctly in all supported browsers.
