Marker Detail - Disease ribbon - Model pop-up

  • US 58 - Marker Detail - disease model pop-up

Test Requirements

  1. Add pop-up page for the "Mouse Models Involving Xxx" column links.
  2. Pop-up should be the same as the pop-up for that marker/disease combination on the Disease Detail page.
  3. Count displayed is typically the number of models present -- regardless if associated via a complex or simple genotype.
    • The exception -- if there are only NOT models -- then the count is the total of NOT models. E.g. Wisp3
  4. The pop-up page lists the all the models for the marker/disease combination and then any NOT models. This is the same as the Disease Detail page.
  5. The models displayed are limited to the mouse gene of the Marker Detail page -- even when the gene may have a mouse paralog. e.g. C4a, C4b
    • C4a has no disease annotations and therefore its Marker Detail page won't show that relationship. Its paralog C4b has one and will display it on its own Marker Detail page.
  6. The pop-up should display correctly on all supported browsers.
