US133 and US124 Test doc

Overview

US133:
  1. creates a new, combined antibody/antigen detail page in the fewi
  2. moves antibody and antigen data into the front-end (fe) database
  3. removes defunct files from old python wi_pg product: antibody detail, antigen detail, and antibody summary (no links to this page anymore)
  4. updates links (on the accession report and the assay detail page) for antibodies and antigens to point to the new page
  5. updates the wiki page listing which pages are hosted by which products
US124:
  1. updates antibody links from the quick search to go to the new, combined antibody/antigen page
  2. removes support for antigen IDs from the quick search

Details

Individual requirements and tests are detailed below. All requirements pass testing.
