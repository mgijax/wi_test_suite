    Note: These links were temporarily removed when we switched to the Hybrid before implementing the new 'HGNC Mouse/Human Homology' page, but were not restored.

Tests

Test Cases

  1. Human Gene in HG and HGNC clusters, HG and HGCN clusters disagree, Hybrid selects HG (link should go to HG cluster page).
  2. Human Gene in HG and HGNC clusters, HG and HGCN clusters disagree, Hybrid selects HGNC (link should go to HGNC cluster page).
  3. Human Gene in HG and HGNC clusters, HG and HGCN clusters agree, Hybrid selects both (link should go to HG cluster page).
  4. Human Gene in HG or HGNC cluster, HG and HGCN clusters disagree, Hybrid selects source that does not include this human gene (reject) (not cases where the human gene is annotated to a disease).
  5. Human Gene in neither HG nor HGNC clusters (double orphan) (no link).
