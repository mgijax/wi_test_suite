GO Marker - show organism for Inferred IDs

  • User Story48 - Show organism in "inferred from" column on GO/Marker page

Test Requirements

  1. Display organism for IDs in the Inferred From column/GO Marker page/Table view. Not all IDs will display an organism - see below for exceptions.
  2. IDs associated to database objects in MGI that include "organism" include MGI, RGD, EC, and some sequence ids.
  3. MGI and RGD IDs will display an organism (per associated marker object)
    • MGI allele ID: organism=mouse
    • RGD gene ID: organism=rat
  4. EC ids will display an organism due to the association with mouse markers in MGI.
  5. Sequence IDs will display organism if associated to an db object in MGI include: EMBL, NCBI, protein_id, RefSeq, and UniProtKB.
    • EMBL sequences (can't find example where organism is displayed)
    • NCBI protein sequences
    • protein_id ...
    • RefSeq protein and transcript sequences: organism = varied
    • UniProtKB protein sequences: organism = varied
  6. Display organism by checking the logical db for both yeast MODs - SGD and PomBase. These IDs are not associated to any MGI db objects.
    • SGD gene ID: organism=budding yeast.
    • PomBase gene ID: organism=fission yeast.
  7. The remaining ID types will not display an organism because they are not associated with any db objects in MGI that has one.
    • CHEBI, GO, InterPro, PANTHER, UniProtKB-KW
  8. If an ID is associated with a sequence and marker object -- the marker object's organism is the preferred one.
  9. If the ID has no object in the MGI database then no organism is displayed.
