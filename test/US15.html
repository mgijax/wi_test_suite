Assign Coordinates to Markers w/no Gene Model

  • User Story 15, Support bulk loading to assign coordinates directly to markers that do not have gene models.

Test Requirements

  • Assign coordinates to any Marker Type.
    • types include: Gene, Pseudogene, DNA segment, Cyto Marker, QTL, BAC/YAC, Other Genome Feature, Complex/Cluster/Region
  • Load coordinates for tRNA genes
  • Load coordinates for multiple markers in one input file
  • Allow curator to specify Genome Build to display in the Prod WI in the Curator input file
  • Allow curator to specify Provider text by marker to display in the Prod WI in the Curator input file
  • Allow curator to override previously assigned coordinates in the Curator input file by marker
  • Allow curator to remove previously assigned coordinates by marker
