US44 - Enable Quick Search Queries for Genotype IDs (TR11952)

Test Requirements
Searches by Productive Genotype* accession IDs (MGI:IDs) in the Quick Search Tool should return the corresponding MGI genotype in the Other Results By ID section.

*Productive Genotype: An MGI genotype annotated to MP terms and/or OMIM terms.

Display Text in the "Other Results By ID section":
Under Type: "MGI Genotype" - This should link to the corresponding genotype page
Under Name/Description: Genotype contains: Marker1, Marker2, ..., MarkerN

Test Environments:
Scrum Bob WI
firien WI

Test Cases/Testing
