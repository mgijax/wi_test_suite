Test Requirements - Phenotype/Disease Query Box

  1. Recognize the following boolean operators: AND, OR, NOT, AND NOT
    • boolean operators must be in upper case; otherwise not recognized.
    • OR NOT is recognized by the software, but there's no plan to document this for users

Tests

Manual tests

  1. Single Gene Queries
    1. PASS Query for marker with no OMIM annotations and no human gene/disease annotations - Query: Bmp3 - Expect: no diseases for marker
    2. PASS Query for marker with no OMIM annotations, but with human gene/disease annotations - Query: Bmp1 - Expect: Osteogenesis Imperfecta, Type Xiii; OI13
    3. PASS Query for marker with OMIM annotation to a disease human only plus MP mouse annotations - Query: MTTP - Expect: All diseases annotated to the human marker MTTP - 2 diseases.
    4. PASS Query for marker with OMIM annotation to a disease mouse and human - Query: 105830 - Expect: All diseases annotated thorugh the genocluster and the human marker - 8 diseases.
    5. PASS Query for marker with OMIM annotation to a disease human only no MP mouse annotations - Query: 300321 - Expect: All diseases annotated to human marker FLNA - 10 diseases.
    6. PASS Query for MP term and have no human diseases returned - Query: MP:0001261 - Expect: No human data brought back - 15 disease clusters
  2. OR queries.
    1. PASS (MP OR MP) - Query: (MP:0000774 OR MP:0001307) - Expect: Only matching on MP terms should not see any human diseases
    2. PASS (MP OR OMIM) - Query: (MP:0000774 OR 211980) - Expect: Both human and mouse diseases
    3. PASS (OMIM OR MP) - Query: (211980 OR MP:0000774) - Expect: Same as previous test
    4. PASS (OMIM OR OMIM) - Query: (211980 OR 311300) - Expect: Both human and mouse diseases, Lung cancer as well as Otopalatodigital Syndrome, Type I. Gene FLNA is added to the list.
  3. AND queries.
    1. PASS (MP AND MP) - Query: (MP:0000774 AND MP:0001307) - Expect: No results, there are no genes that have both annotations
    2. PASS (MP AND MP) - Query: (MP:0002006 AND MP:0005395) - Expect: Mouse only diseases showing
    3. PASS (MP AND OMIM) - Query: (MP:0002006 AND 155255) - Expect: Only mouse diseases showing
    4. PASS (OMIM AND MP) - Query: (155255 AND MP:0002006) - Expect: Same as previous test
    5. PASS (OMIM AND OMIM) - Query: (211980 AND 260350) - Expect: Each term has both mouse and human, but anded together have only human diseases.
    6. PASS (OMIM AND OMIM) - Query: (211980 AND 131200) - Expect: Each term has both mouse and human, but anded together have only mouse diseases.
    7. PASS (OMIM AND OMIM) - Query: (311300 AND 300049) - Expect: Each term has both mouse and human, but anded together match on both the mouse and human side.
    8. PASS (OMIM AND OMIM) - Query: (211980 AND 300321) - Expect: Each term has both mouse and human, but anded together have no results.
  4. AND NOT queries.
    1. PASS (MP AND NOT MP) - Query: (MP:0006126 AND NOT MP:0010402) - Expect: Phenotypes for "abnormal outflow tract development" but not "ventricular septal defect" and only mouse diseases showing
    2. PASS (MP AND NOT OMIM) - Query: (MP:0006126 AND NOT 188400) - Expect: Phenotypes for "abnormal outflow tract development" but no diseases for "DiGeorge Syndrome; DGS" and only mouse diseases showing
    3. PASS (OMIM AND NOT MP) - Query: (188400 AND NOT MP:0006126) - Expect: Diseases for ""DiGeorge Syndrome; DGS" but no phenotypes for "abnormal outflow tract development" and human and mouse diseases showing
    4. PASS (OMIM AND NOT OMIM) - Query: (309350 AND NOT 305620) - Expect: Mouse Diseases only 305620 matches on human side of FLNA and removes all human Diseases
    5. PASS (OMIM AND NOT OMIM) - Query: (125853 AND NOT 245590) - Expect: Human Diseases only 245590 matches on mouse side of HNF1A and removes all mouse Diseases
    6. PASS (OMIM AND NOT OMIM) - Query: (105830 AND NOT 300260) - Expect: Human and Mouse minus 300260 which matches on mouse side of UBE3A and removes those results
    7. PASS (OMIM AND NOT OMIM) - Query: (305620 AND NOT 309350) - Expect: No Diseases 309350 matches on both human and mouse side of FLNA and removes all Diseases
    8. PASS (OMIM AND NOT OMIM) - Query: (311300 AND NOT 300049) - Expect: No results because both diseases match on human side of FLNA
  5. Complex queries.
    1. PASS (MP OR MP) AND NOT OMIM - Query: ((MP:0000774 OR MP:0001307) AND NOT 608629)- Expect: Same results as query 2.1 minus "Joubert Syndrome 3; JBTS3" disease, and with no human data
    2. PASS (OMIM OR OMIM) AND NOT MP - Query: ((211980 OR 311300) AND, NOT, MP:0004842) - Expect: Same results as query 2.4 minus "abnormal large intestine crypts of Lieberkuhn morphology" phenotype, and showing both human and mouse data
    3. PASS MP OR (MP AND NOT OMIM) - Query: (MP:0000774 OR (MP:0006126 AND NOT 188400)) - Expect: Same results as query 4.2 plus "decreased brain size" phenotype, and with no human data
    4. PASS OMIM OR (MP AND NOT OMIM) - Query: (311300 OR (MP:0006126 AND NOT 188400)) - Expect: Same results as query 4.2 plus "Otopalatodigital Syndrome, Type I; OPD1" disease, and with human data
