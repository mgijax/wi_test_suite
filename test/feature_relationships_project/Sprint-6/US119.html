Allele Detail - Mutation Involves changes

  • User Story 119 - Implement allele detail changes for mutation involves

Contents: Title | Test Requirements | Manual Tests | SQL Tests

Test Requirements

  1. Add teaser of Mutation Involves relationships in the Mutation Description ribbon
  2. Display the total number of "related" markers
  3. Display up to 3 markers as a teaser for the user -- if more than 3 markers are related include an ellipsis.
  4. Link markers in the teaser list to their Marker Detail page.
  5. Order the markers in the teaser list in location order.
  6. Display a "View All" button that when clicked displays the new pop-up page of related markers.
  7. Wrap new text as needed when a Molecular Image is present for the mutation.
  8. Display consistently in all supported browsers (PC - Chrome, Firefox; Mac - Chrome, Firefox, Safari)
