Mutation Involves - Allele Summaries

  • User Story 106 - Change Allele Summaries - mutation involves

Contents: Title | Test Requirements | Manual Tests | SQL Tests

Test Requirements

  1. Identify alleles that are an "orginator" in mutation involves relationships and modify the display of its row in all Allele Summary reports (including when generated from QF)
    1. Add mutation involves teaser list of markers to the Category column (same rules as Allele Detail)
    2. Add link to new mutation involves pop-up page with a "View All" button (same button style as Allele Detail)
    3. Display this new information after all existing fields in the Category column (allele types, attributes)
  2. New pop-up page linked to has same content and style as when linked to from the Allele Detail page
  3. All existing Allele Summary variations on the Marker Detail page should include the mutation involves alleles as appropriate
    1. The "All" alleles link should include any mutation involves alleles present.
    2. Specific categories (e.g. targeted) should include mutation involves alleles if that allele falls in that category.
  4. Create a new Allele Summary version that only returns mutation involves alleles for a marker.
    • Display a new sub-heading for this Summary version that informs users that only mutation involves alleles have been returned
  5. New features should work properly on all supported browsers.
  6. No changes to the Text and Excel output files are intended as part of this user story.
