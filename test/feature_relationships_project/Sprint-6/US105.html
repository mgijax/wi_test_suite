Mutation Involves - new pop-up page

  • User Story 105 - Mutation Involves: new pop-up page

Contents: Title | Test Requirements | Manual Tests | SQL Tests

Test Requirements

  1. Display mutation involves relationships in a separate browser window from where linked
  2. Use mockup as a guide for display, headings, logos, etc
  3. Pagination is required as the list of relationships may be lengthy
  4. Include the following columns in the table
    1. Mutation - display allele symbol
    2. Effects on Gene/Genome Feature - display appropriate term synonym and marker symbol
    3. Gene/Genome Feature Type
    4. Gene/Genome Feature Location - chromosome, coordinates, and strand (when present)
    5. Reference - J number
    6. Notes
  5. Link Marker Symbol and Reference to appropriate detail page
  6. Sort relationships by location
  7. Display consistently in all supported browsers (PC - Chrome, Firefox; Mac - Chrome, Firefox, Safari)
