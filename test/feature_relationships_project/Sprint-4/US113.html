Test Requirements

  1. Provide a link to the new user help doc via the "?" icon for the Explorer (graph and table view)
    • "INTERACTION_explorer_help.shtml".
  2. Provide a pop-up to the user for specific help re: the graph controls/display (pan, zoom, etc.)
    • Use the paradigm used on the Homology Class page
    • Text for pop-up - David S. to provide.
  3. Provide users a way to tell the difference between predicted and validated relationships.
    1. Validated = blue
    2. Predicted = red
    3. Any other value (blank, etc) treated as Predicted.
  4. Provide users ability to zoom in and out
    1. Display +/- for users to click on to zoom
    2. Provide zooming using mouse wheel
  5. Provide the graph control features for all supported browsers. Controls limited to Pan and Zoom.
    • PC: Firefox; Chrome & MAC: Firefox; Chrome; Safari
