Test Requirements

  1. Update load software to recognize & load the relationship category - "interacts_with"
  2. Update load software to recognize & load the following input fields that may be present for any category (not just interacts_with)
    1. Property fields
    2. Notes
  3. Validate the Property fields' column headings.
    1. Required prefix = Property:
    2. The text that follows the prefix must be present in the "Relationship Properties" controlled vocabulary
    3. There are no Property fields that are required to be present in the input file.
    4. Display the following fields when this error is found: xxx,xxx,xxx
    5. If this error is found during the load process the load will abort.
  4. Validate the data in the Property:score column (if the column is present)
    1. Valid values for score = blank, an integer, or a float
    2. plus (+) or minus (-) are allowed, but not required
    3. Display the following fields when this error is found: xxx,xxx,xxx
    4. If this error is found during the load process the load will abort.
  5. During the QC process create the following files:
    1. file 1
    2. file 2
    3. etc.
  6. Accept & load a file that contains both interacts_with and cluster relationships.
