SNP Summary - Strain columns

  • US61: Add strain columns to SNP Summary

Test Requirements

  1. Add Strain columns to the SNP Summary -- both from the Query Form and Marker Detail.
  2. Insert strain columns between Variation Type and Allele Summary columns.
  3. Sort the strains returned in Smart Alpha order; no concept of primary/secondary strain lists anymore.
  4. Use CSS rotation for strain column headings.
  5. Strain name in column heading uses angle brackets for superscripts; also truncated.
  6. Roll over tool tip for strain columns - tool tip recognizes superscripts and is not truncated.
  7. Alleles displayed use standard colors for ACGT.
  8. Support ? and underline (_) in allele cells.
  9. Only SNP Variations are returned in the SNP Summary -- so the IN-DEL and Mixed features are not part of this story.
  10. No query of strains in this user story -- so all strains that are present in the set of query results are displayed.
