Gene Nomen - search by comma delim lists

  • US57: GXD QF: allow comma-delimited list in nomen field

Test Requirements

  1. Accept comma delimited lists of strings in the gene nomenclature box.
  2. Treat each comma delimited element the same as if a single token was entered. Merge the search results.
  3. Search rules -- applies for searches without a comma or to a string delimited by commas
    1. Search marker symbols, names, and synonyms
    2. Allow wild cards: *token, token*, *token*
    3. ignore special characters (hyphen, quotes, etc)
    4. multiple tokens in a string - token1 AND token2
  4. If any element of the comma delimited list does not return results -- no error or warning message is displayed.
  5. If all of the elements of the comma delimited list do not return results -- no error or warning message is displayed.

Tests

Manual tests

  1. Verify that single token items in a comma delimited list search symbols, names, and synonyms

    new test

    1. Enter a list of tokens that match current symbols: xxx,xxx,xxx. Expect: N matches to gene symbols.
    2. Enter a list of tokens that match current names: xxx,xxx,xxx. Expect: N matches to gene names.
    3. Enter a list of tokens that match current synonyms: xxx,xxx,xxx. Expect: N matches to gene synonyms.
    4. Enter a token that matches allele nomenclature: xxx,xxx. Expect: 0 matches to allele nomenclature.
    5. Enter a token that matches orthologs: Rg, kit, dlx5. Expect: 0 matches by ortholog nomenclature (Rg = human synonym (mouse C4a, C4b); Dlx5 = Rat synonym (mouse Dlx2).
  2. Verify that multiple token items in a comma delimited list search symbols, names, synonyms

    new test

    1. Enter a multiple token element that matches symbols: xxx-xxx,xxx,xxx. Expect: N matches to gene symbols.
    2. Enter a multiple token element that matches names: xxx xxx,xxx,xxx. Expect: N matches.
    3. Enter a multiple token element that matches synonyms: xxx-xxx, xxx, xxx. Expect: N matches.
  3. Verify that wild-cards are accepted in comma delimited lists.

    new test

    1. Enter a token with a preceding wild card; xxx, *xxx, xxx. Expect: N matches.
    2. Enter a token with a following wild card; xxx, xxx*, xxx. Expect: N matches.
    3. Enter a token enclosed by wild cards; xxx, *xxx*, xxx. Expect: N matches.
    4. Enter a multiple token element with wild cards; xxx, xxx* xxx*, xxx. Expect: N matches.
  4. Verify special characters are ignored in comma delimited lists

    new test

    1. Enter a token with a apostrophe; xxx, xxx'x, xxx. Expect: N matches.
