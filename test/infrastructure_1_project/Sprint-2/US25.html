Rewrite MP Annotation Page - term for a marker

  • User Story 25 - Rewrite MP Annotation Page - term for a marker

Contents: Title | Test Requirements | Manual Tests | SQL Tests

Test Requirements

  1. The report keeps the same rules for what data is included on the page
    1. Data = MP annotations for the selected term and its sub-terms for a specific marker.
    2. Data = normal annotations are included
  2. The report displays the same data and links on the page
    1. Headings = title; marker heading bar (symbol, name, id); genotype/annotation count; searched term
    2. Columns displayed = Allelic Composition (Genetic Background), Annotated Term, Reference (may be more than 1)
    3. Links = alleles, MP terms, references, marker
  3. The report uses the same display rules on the page
    1. Genotype display = recognize rules for line breaks set by the curators
    2. Genotypes are listed just once and may include multiple annotations to the term (and/or child terms) if present
    3. Sort order = Primary: alphanumeric by the genotype; Secondary: alphanumeric by term
  4. Update links to this page in Batch Query
  5. Add URL to list of forwarded pages or not?
  6. Display correctly on all supported browsers (IE, firefox, chrome, safari)
  7. Add pagination?

Manual tests

  1. Verify the correct MP Annotation data is included on the page

    new test

    1. Display MP Annotation page for a term/marker with annotated subterms, Gene: cw and Term: MP:0000367 (abnormal coat/hair morphology). Expected results: 2 genos/4 annots
    2. Display MP Annotation page for a term with annotated subterms, multiple annotations for a genotype, and complex genotypes, Gene: Lepr and Term: MP:0000003 (abnormal adipose tissue morphology. Expected results: 28 genos/53 annots
    3. Display MP Annotation page for a term with no annotated subterms, Gene: Trp53 and Term MP:0000160 (kyphosis). Expected results: 3 genos/3 annots
    4. Display MP Annotation page for a term with multiple annotations for a genotype, Gene: Brca2 and MP:0000628 (abnormal mammary gland development). Expected results: 2 genos/3 annots
    5. Display MP Annotation page for a term that includes normal annotations - this case doesn't exist as these normal annotations are not included in Batch Query results
    6. Display MP Annotation page for a Normal phenotype term, Gene: Aco1 and MP:0002169 (no abnormal phenotype detected). Expected results: 2 genos/2 annots
    7. Display MP Annotation page for a term with only single annotations for all genotypes, Gene: Sry and Term MP:0002995 (primary sex reversal). Expected results: 6 genos/6 annots
    8. Display MP Annotation page for a term with only 1 annotation, Gene: Bmp4 and MP:0000024 (lowered ear position). Expected results: 1 genos/1 annots
