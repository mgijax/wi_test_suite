Test Requirements

  1. Maintain GXD Filter Functions
    Filters will be tested separately and in combination (to limit result counts)
    1. Gene Symbol
    2. MGI Gene ID
    3. Theiler Stage

    5. Expression

    7. Specimen Mutated In: Gene Symbol

    9. Reference ID
    10. MGI Probe ID
  2. Maintain GXD Attributes
    Attributes will be tested simultaneously with Filter tests
  3. Maintain Integration Between GXD and MGD BioMarts
    Under "CHOOSE ADDITIONAL DATASET", filters can be applied to the MGD BioMart, which should limit GXD BioMart query results.
