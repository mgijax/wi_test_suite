Test Requirements

  1. Add Acc ID query box next to Quick Search box on Prod WI toolbar
    1. This is the Python WI version that has been in place for curators in the Curator Webtools product
    2. Existing limitations of the Acc ID Query will continue...
    3. No plan to include recent changes to the fewi "Accession" functionality made in 5.13
  2. Maintain existing Acc ID query functionality that is now available on Curator Webtools page
    1. Results returned for public and private MGI ids (but not ones in nomenclature table)
    2. Results returned for objects just entered into the database by curator (no waiting)
    3. List of comma delimited IDs accepted
    4. Direct links to web page when only a single object is matched
    5. Results returned for MGI id of withdrawn markers
