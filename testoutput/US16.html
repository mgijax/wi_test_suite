Assign Coordinates to Markers with Gene Models

  • User Story 16, Support bulk loading to assign coordinates directly to markers that have gene models (i.e. they have coordinates assigned via the Gene Model/Representative Sequence process).

Test Requirements

  • Assign coordinates to markers with Gene Models
    • marker types include: Gene, Pseudogene
    • gene model providers include: VEGA, Ensembl, NCBI
  • Load coordinates for multiple markers in one input file
  • Allow curator to specify Genome Build to display in the Prod WI in the Curator input file
  • Allow curator to specify Provider text by marker to display in the Prod WI in the Curator input file
  • Allow curator to override previously assigned coordinates in the Curator input file by marker
  • Allow curator to remove previously assigned coordinates by marker making the Gene Model coordinates active again

Tests

Manual tests

  1. Loaded chr, coordinates, provider text, and build text are displayed in the Sequence Map section of the Production WI Marker Detail page for markers with Gene Models.

    new test

    1. Display marker xxx. This marker has a marker type of Gene
      • Verify the Sequence Map display:
        chrNN:nnn-nnn bp, strand
        (From xxx annotation of NCBI Build XX)
    2. Display marker xxx. This marker has a marker type of Pseudogene
      • Verify the Sequence Map display:
        chrNN:nnn-nnn bp, strand
        (From xxx annotation of NCBI Build XX)
  2. Verify newly assigned coordinates override previous coordinates defined by Gene Model.

      new test

    1. Display marker xxx. This marker has a gene model provider of VEGA
      • Verify the Sequence Map display:
        chrNN:nnn-nnn bp, strand
        (From xxx annotation of NCBI Build XX)
    2. Display marker xxx This marker has a gene model provider of Ensembl
      • Verify the Sequence Map display:
        chrNN:nnn-nnn bp
        strand(From xxx annotation of NCBI Build XX)
    3. Display marker xxx. This marker has a gene model provider of NCBI
      • Verify the Sequence Map display:
        chrNN:nnn-nnn bp
        (From xxx annotation of NCBI Build XX)
  3. Verify assignment of coordinates with special cases in re: to chromosomes

      new test

    1. Display marker xxx. This marker is on the MT chromosome.
      • Verify the Sequence Map display:
        chrMT:nnn-nnn bp, + strand
        (From xxx annotation of NCBI Build XX)
  4. Markers with no coordinate assignment via the load continue to display gene model coordinates and credit/build text. (as-of xxx - coordinates on Public WI for the following markers)

      new test

    1. Display marker xxx. This marker has a marker type of Gene & provider of VEGA.
    2. Display marker xxx. This marker has a marker type of Pseudogene & provider of NCBI.
    3. Display marker xxx. This marker has a marker type of Gene & provider of Ensembl.
